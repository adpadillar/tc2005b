1. Understanding the basics

1.1 Head and body

When you’re building a website, you’re working with two main parts: the head and the body. The head is like the brain of the website, where you put all the important stuff that doesn’t show up on the page. This includes the title of the page, links to stylesheets, and meta information like keywords and descriptions.

The body is where all the content of the page goes. This is where you put your text, images, videos, and anything else you want people to see when they visit your site.

1.2 Headings

Headings are an important part of any website. They help to break up the content and make it easier to read. There are six levels of headings in HTML, with <h1> being the most important and <h6> being the least important.

1.3 Images

Images are a great way to make your website more visually appealing. You can use them to show off your products, illustrate your blog posts, or just add some color to your site. To add an image to your website, you use the <img> tag.

1.4 Links

Links are what make the web work. They’re what allow you to move from one page to another, from one website to another, and from one part of a page to another. To create a link in HTML, you use the <a> tag.

2. Javascript

2.1 Variable types

In Javascript, there are several ways to declare a variable. You can use the var keyword, the let keyword, or the const keyword. Each of these has its own use cases and rules for how it can be used.

The var keyword is the oldest way to declare a variable in Javascript. It has some quirks that can make it difficult to work with, so it’s generally recommended to use let or const instead.

The let keyword was introduced in ES6 as a replacement for var. It has some advantages over var, such as block scoping and the ability to reassign values.

The const keyword is used to declare a variable that can’t be reassigned. This can be useful when you want to make sure that a variable doesn’t change throughout your program.
